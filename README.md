# Obeys Agency Website Clone

This is a clone of the Obeys Agency website, built using HTML, CSS, and JavaScript. The project is deployed on Netlify for easy access.

## 🚀 Live Demo
Check out the live version here: [Deployed on Netlify] ( https://playful-begonia-a4f9f5.netlify.app/ )

## 📌 Features
- Interactive animations
- Smooth scrolling
- Optimized for performance
- Cross-browser compatibility

## 🛠️ Technologies Used
- **HTML** - Structure of the website
- **CSS** - Styling and animations
- **JavaScript** - Interactivity and functionality

## 📂 Project Setup
Open `index.html` in your browser or use Live Server in VS Code.
